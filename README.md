Analisis rapido para pracicar metodologias de analisis de datos con Python, usando los modulos de Pandas, Seaborn, Matplot y SKlearn.

Todos los datos son inventados y no vienen de personas reales. Se usan a modo demostrativo para poner en practica y mostrar un trabajo de Analisis de un dataseet.
